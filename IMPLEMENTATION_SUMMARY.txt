MARKDOWN SUPPORT
================================================================================

The _md_to_html() method converts basic markdown:
  **text**      →  <strong>text</strong>
  - item        →  <ul><li>item</li></ul>
  * item        →  <ul><li>item</li></ul>
  1. item       →  <ol><li>item</li></ol>
  ### Header    →  <h3>Header</h3>
  \n\n          →  paragraph break
  \n            →  <br> tag

Algorithm:
  1. HTML-escape all input first
  2. Process line-by-line
  3. Track list state (type, open/close)
  4. Handle headers, bold, and newlines
  5. Return clean, safe HTML
